# US Fencing Divisions Interactive Map

An interactive React map visualization of US Fencing divisions with bubble markers sized by different metrics.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser to the URL shown in the terminal (typically `http://localhost:5173`)

## Features

- Interactive map with clickable markers for each fencing division
- Toggle between different metrics:
  - Total Members
  - Foil Rated
  - Epee Rated
  - Saber Rated
- Bubble size represents the selected metric value
- Popup with detailed statistics for each division

## Technologies

- React
- React Leaflet
- Leaflet
- Tailwind CSS
- Vite
